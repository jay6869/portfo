This project is the most hardware-heavy thing I've done, and the one that most changed how
I think about "trusted" sensor inputs. CSI is treated as ground truth by a growing number
of products — and ground truth is exactly the kind of assumption worth attacking.

## The setup

Two ESP32-S3 boards: one passive sniffer, one crafted-frame transmitter, both disciplined
against the same AP beacon so injected frames land in a predictable timing window. A laptop
captures CSI snapshots for offline comparison in NumPy.

## Reading the result

The clearest evidence was visual — a spectrogram of a genuine "person present" event next
to a spoofed one, close enough that the commercial demo's classifier called both positive.

<Callout>
I worked with the affected vendor before publishing implementation specifics. Exact frame
payloads and timing offsets are intentionally omitted here, and all capture was done on
hardware and spectrum I control.
